== Description ==

This little plugin automatically set topics status older than a month to resolved. This plugin requires BBpress and BBps Support Forum

== Installation ==

1. Upload the whole unzipped folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
